    - Does not mutate original object

    Export `sanitizeUserInput(data: Record<string, unknown>): Record<string, unknown>`:
    - Sanitizes all string fields in the object

    - Applies to: `displayName`, `description`, `notes`, `title`, `body`, `reason`, `summary`

    Export `sanitizeReportInput(data: Record<string, unknown>): Record<string, unknown>`:

    Create `functions/src/security/rateLimit.ts`:
    - Import `admin` from `firebase-admin`
    - Import `getFirestore` from `firebase-admin/firestore`

    **Rate limit configuration:**
    ```typescript
    const DEFAULT_RATE_LIMIT = {
      maxReports: 5,
      windowMs: 60 * 60 * 1000, // 1 hour
    }

    const SURGE_RATE_LIMIT = {
      maxReports: 20,
      windowMs: 60 * 60 * 1000, // 1 hour
    }
    ```

    **Rate limit state storage:**
    - Store rate limit counters in `rate_limits/{userId}` document
    - Fields: `count`, `windowStart` (timestamp), `surgeMode`, `surgeModeExpires`

    Export `checkRateLimit(userId: string, isAdminRequest: boolean = false): Promise<{ allowed: boolean; remaining: number; resetAt: Date }>`:
    - Get rate limit document for user
    - Check if current time is past windowStart + windowMs (reset if expired)
    - If surge mode is active (surgeModeExpires > now), use SURGE_RATE_LIMIT
    - Otherwise use DEFAULT_RATE_LIMIT
    - If count >= limit, return `{ allowed: false, remaining: 0, resetAt }`
    - If allowed, return `{ allowed: true, remaining: limit - count - 1, resetAt }`

    Export `incrementRateLimit(userId: string): Promise<void>`:
    - Atomically increment count in rate_limits document
    - Uses Firestore transaction to prevent race conditions

    Export `setSurgeMode(municipalityCode: string, enabled: boolean, durationMs: number = 24 * 60 * 60 * 1000): Promise<void>`:
    - Called by admin to enable/disable surge mode
    - Updates `rate_limits/surge/{municipalityCode}` document
    - When enabled: all users in that municipality get surge rate limit
    - Only municipal_admin or superadmin can call this

    Export `isSurgeModeActive(municipalityCode: string): Promise<boolean>`:
    - Checks surge document for municipality
    - Returns true if surge is active and not expired

    **Firestore structure for rate limits:**
    ```
    rate_limits/{userId}
      - count: number
      - windowStart: Timestamp
      - surgeMode: boolean
      - surgeModeExpires: Timestamp | null

    rate_limits/surge/{municipalityCode}
      - enabled: boolean
      - expiresAt: Timestamp
      - enabledBy: string (admin uid)
    ```

    Create `functions/src/security/validateAuth.ts`:
    - Import `functions` from `firebase-functions`
    - Import `isSuperadmin, isMunicipalAdmin` from `../auth/claims`

    Export `validateSuperadmin(context: functions.https.CallableContext): void`:
    - Checks `context.auth` exists
    - Checks `isSuperadmin(context.auth.token)`
    - Throws `functions.https.HttpsError('permission-denied', 'Superadmin access required')` if not

    Export `validateMunicipalAdmin(context: functions.https.CallableContext, municipalityCode: string): void`:
    - Checks `context.auth` exists
    - Checks `isSuperadmin(context.auth.token)` OR `isMunicipalAdmin(context.auth.token, municipalityCode)`
    - Throws `functions.https.HttpsError('permission-denied', 'Admin access required for this municipality')` if not

    Export `validateAuthenticated(context: functions.https.CallableContext): void`:
    - Checks `context.auth` exists
    - Throws `functions.https.HttpsError('unauthenticated', 'Authentication required')` if not

    Export `validateWriteScope(context: functions.https.CallableContext, dataMunicipalityCode: string | null, resourceMunicipalityCode: string | null): void`:
    - Superadmin can write anywhere
    - Municipal admin can only write to their municipality
    - If `dataMunicipalityCode` doesn't match caller's scope, throw permission-denied
    - For reports: `dataMunicipalityCode` comes from the report being created

    **Usage in Cloud Functions:**
    ```typescript
    import { validateMunicipalAdmin, validateWriteScope } from './security/validateAuth'

    export const createReport = onCall(async (request) => {
      validateAuthenticated(request) // Must be logged in

      const data = request.data
      const municipalityCode = data.municipalityCode

      // Check rate limit
      const rateCheck = await checkRateLimit(request.auth.uid)
      if (!rateCheck.allowed) {
        throw new functions.https.HttpsError('resource-exhausted', 'Rate limit exceeded')
      }

      // Validate write scope
      validateWriteScope(request, municipalityCode, null)

      // Sanitize input
      const sanitized = sanitizeReportInput(data)

      // Create report...
      await incrementRateLimit(request.auth.uid)
    })
    ```
