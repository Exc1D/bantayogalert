# Features

D-01: Source code organized under `src/features/{feature}/` with co-located components, hooks, types, and queries.

## Structure

Each feature directory contains:
- `components/` — Feature-specific React components
- `hooks/` — Custom React hooks
- `types/` — Feature TypeScript interfaces and types
- `queries/` — TanStack Query hooks and API calls

## Features

- **auth/** — Authentication (Phase 3)
- **reports/** — Report submission and tracking (Phase 5)
- **map/** — Map display and markers (Phase 6)
- **alerts/** — Announcements and push notifications (Phase 10)
- **contacts/** — Responder contacts management (Phase 8)
- **analytics/** — Analytics dashboards (Phase 11)
- **admin/** — Admin triage workflows (Phase 9)
